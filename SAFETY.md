# LISA Safety Notice

**Version:** 1.1 local testing build

## Assistive tool — not a medical device

LISA is an **assistive communication tool** designed to help people communicate using intentional eye movements. It is **not a certified medical device** at this stage.

## Supervised testing required

- **Test with supervision** before relying on LISA in daily life.
- Practice wink sequences, confirmation, and speech timing in a calm environment first.
- Use **Menu → Testing Checklist** to verify each capability before wider testing.

## Emergency limitations

- **Do not rely on LISA as your only emergency system yet.**
- Emergency alarm and caregiver notification features are **local-only** in this build — no real phone alerts are sent.
- Emergency features **must be tested with caregivers present** before any real-world dependency.
- Always maintain alternative emergency methods (call button, caregiver presence, medical alert systems).

## Reset and alarm

- The **Reset** button stops the local emergency alarm and clears the current session.
- Confirm that Reset works during testing (see Testing Checklist).

## When to stop testing

Stop using this build and seek professional guidance if:

- Wink detection is consistently unreliable for the user
- Speech timing causes confusion or distress
- Emergency behavior is unpredictable

## Medical decisions

LISA does not diagnose, treat, or monitor medical conditions. All care decisions remain with the user, caregivers, and qualified healthcare professionals.
