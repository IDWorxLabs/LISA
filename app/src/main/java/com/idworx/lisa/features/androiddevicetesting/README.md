# Android Device Testing Protocol V1 (in-app module)

See also: `features/android-device-testing/README.md`

Defines repeatable real Android device testing. All steps default to NOT_TESTED until evidence is recorded.

Run `./gradlew validateAndroidDeviceTestingProtocolAuthorityV1` to emit `ANDROID_DEVICE_TESTING_PROTOCOL_AUTHORITY_V1_PASS`.
