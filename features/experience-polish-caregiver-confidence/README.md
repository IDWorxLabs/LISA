# LISA Caregiver Confidence V1

Help caregivers feel confident **setting up, supporting, and troubleshooting** Lisa — without turning the app into a technical tool.

## Support principles

| Focus | Behavior |
|-------|----------|
| Phone positioning | Steady, eye-level placement hints during setup |
| Camera / lighting | Plain guidance for even light, no backlight |
| Calibration support | Dot-by-dot progress + what to do if quality is poor |
| Caregiver-only hints | Visible strips — not spoken over the user |
| Troubleshooting | Sensitivity and recalibration in everyday language |
| Progress visibility | Lesson counts and successes on coach strip |
| Tracking recovery | Face-lost recovery with repositioning steps |
| What to do now | One clear next action per setup step |
