# LISA Patient Communication Coach V1

Lisa teaches communication like a **patient human coach** — not a drill instructor.

## Coaching principles

| Focus | Behavior |
|-------|----------|
| Lesson pacing | Brief pause between lessons; slower narration when struggling |
| Phrase progression | Daily phrases first (Levels 1–2), then building/advanced |
| When to repeat | After 2 missed attempts — Lisa repeats the phrase calmly |
| When to celebrate | Tier 1 every success; tier 2 at level milestones |
| When to slow down | After 3 consecutive misses — patience prompts + longer pauses |
| Fatigue-friendly | Rest suggestion every 5 lessons |
| Caregiver progress | Visible strip: lesson X of Y, level, successes, coach hint |
| Gesture jumps | Catalog enforces max +1 difficulty level between adjacent lessons |

## Gestures unchanged

Brain 1 interaction standard (L2 primary, R2 secondary, L1 R1 confirm, R1 L1 cancel) is unchanged.
Coach logic only affects **timing, narration, and UI hints**.
