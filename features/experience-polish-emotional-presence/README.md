# LISA Emotional Presence V1

Lisa feels **emotionally present** without becoming an AI chatbot.

## Presence principles

| Focus | Behavior |
|-------|----------|
| Warm return greetings | Up to 2 lines when a returning user opens a session |
| Calm session openings | 1 line for first-time calm open |
| Long-pause encouragement | One gentle line after 45s idle during a wink sequence |
| Caregiver reassurance | Personality Engine lines visible in coach hints |
| Fatigue check-ins | Brief rest acknowledgment when coach suggests a break |
| Emotional milestones | Up to 2 lines on major coach milestones |
| No overtalking | Max 8 presence lines/hour; 90s minimum between speeches |
| No fake intelligence | Only observable state — never inferred emotions |
| Single voice | All dialogue through Personality Engine tags |
