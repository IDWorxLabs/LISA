# LISA Learning Journey V1

The official Brain 1 first-time user experience — seven stages before the permanent Communication Workspace.

## Stages

1. **Meet Lisa** — trust-building conversation (Zero-Touch)
2. **Getting Ready** — camera, face detection, environment
3. **Calibration** — conversational eye calibration
4. **Communication Fundamentals** — ~20 essential phrases
5. **Communication Mastery** — mixed random recall practice
6. **Communication Workspace Navigation** — 8 progressive workspace lessons (emergency last)
7. **LISA Certified Communicator** — certification and permanent workspace access

## First install (caregiver only)

- Start Guided Learning
- Skip to Communication Workspace (with strong recommendation confirmation)

After certification, future launches open directly into the Communication Workspace.

## Practice (menu)

- Replay Learning Journey
- Practice Communication
- Practice Workspace Navigation
