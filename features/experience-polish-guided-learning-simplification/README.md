# LISA Guided Learning Simplification V1

Guided Learning is **Lisa personally teaching** — not a training dashboard.

## Three questions only

Every screen answers:

1. What is Lisa saying?
2. What should I do?
3. What happens next?

## What was removed from Guided Learning

- Lesson numbers and progress bars
- Success counters and statistics
- Caregiver strips during lessons
- Pause / Repeat / Skip narration controls
- Practice Again / Reduce Sensitivity buttons
- Dashboard-style cards and clutter

## What remains

- **Welcome to Lisa** — mandatory first launch with two calm choices
- **Lesson screen** — large phrase, friendly gesture label, one instruction
- **Lisa-controlled progression** — coach logic runs silently; Lisa speaks and advances
